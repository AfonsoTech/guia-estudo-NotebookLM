# Introdução
Treinei uma IA do zero da ferramenta NotebookLM com fontes de dados diversas como videos, textos e imagens e especificas para que permita o usuario usufruir de uma conversa inteligente, podcast, video, slides e mapa mental com o objetivo de guiar seu estudo sobre as tendencias da Tecnologia da Informação para os proximos anos.

# Fluxo de trabalho
1. Criei um novo caderno
2. Adicionei fonte de dados texto com o prompt:
  " A partir de agora, sua função é ser um Analista de Inteligência de Mercado em TI. Seu objetivo é analisar todos os documentos que eu enviar, cruzar as informações e extrair exclusivamente as novas atualizações, lançamentos, sinais de mudança e tendências emergentes da Tecnologia da Informação. Gere resumos práticos com foco no que há de mais inovador."
3. Realizei uma pesquisa profunda(deep research) com o seguinte prompt:
   "Quais são as principais tendências emergentes no mercado de TI."
   OBS: Ele buscou as 10 principais fontes confiaveis na pesquisa
4. Importei as fontes pesquisadas
5. Treinei o chat com os respectivos comandos:
   "Discuss what these sources say about Tecnologias Emergentes, in the larger context of Tendências e Inovações de TI para 2025."
   "Discuss what these sources say about IA para detecção proativa de ameaças, in the larger context of Cibersegurança e Confiança Digital."
    OBS: as respostas foram positivas
6. Pedi ao Estudio para gerar um video, podcast, mapa mental e slides resumo.
7.  Testei cada resultado
    OBS: foram todos positivo
